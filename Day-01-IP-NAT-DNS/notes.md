# Day 1 – IP Addressing, NAT & DNS

## What I Learned

- IP address identifies a device on a network.
- Private IP is used inside local networks.
- Public IP is used on the internet.
- NAT translates private IP to public IP.
- DNS converts domain names into IP addresses.

## Website Access Flow

1. User types google.com
2. DNS resolves domain to IP address
3. Request goes through NAT
4. Public IP communicates with server
5. Server responds back
