# Day 3 – OSI Model

## 7 Layers

7. Application
6. Presentation
5. Session
4. Transport
3. Network
2. Data Link
1. Physical

## Key Understanding

- Layer 3 handles IP addressing.
- Layer 4 handles TCP/UDP and ports.
- Firewalls can filter traffic based on Layer 3 or Layer 4.
- Tools like Nmap scan ports at Layer 4.
