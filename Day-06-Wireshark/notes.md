# Day 6 – Packet Sniffing & Wireshark

## What I Learned
- Packet sniffing is the process of capturing network packets.
- Wireshark is a tool used to analyze network traffic.
- It allows us to see protocols, source IP, destination IP, and ports.

## Key Concepts
- Network packets contain small pieces of data.
- Packet analysis helps detect suspicious network activity.

## Tool
Wireshark – Network protocol analyzer used by cybersecurity professionals.
