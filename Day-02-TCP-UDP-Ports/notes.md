# Day 2 – TCP, UDP & Ports

## TCP
- Reliable
- Uses 3-way handshake
- Uses acknowledgements
- Retransmits lost packets

## UDP
- Faster
- No acknowledgements
- No retransmission
- Used in streaming and gaming

## Ports
- Identify specific services on a device.
- Example:
  - 80 – HTTP
  - 443 – HTTPS
  - 22 – SSH
  - 53 – DNS

## 5-Tuple
A network session is identified by:
- Source IP
- Source Port
- Destination IP
- Destination Port
- Protocol
